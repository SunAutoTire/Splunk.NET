SunAuto.Logging requires the following JSON to be added to the "Logging" object in the appsettings.json
e.g.,

"SunAuto": {
	"Application": <ApplicationName>,
	"Path": <Logfile Path if using file option>,
	"Environment": <Local, Development, Staging, or Production>,
	"BaseUrl": <http://localhost:7235>,
	"ApiKey": <API Key from Storage Account,
	"LogLevel": {
		"Default": <Trace, Debug, etc.>,
		"Microsoft.Hosting": <Trace, Debug, etc.>
	}
},
